# Deckbound — Name Bank

> **Authored flavor reference — living document.** A curated pool of setting names
> (locations, adventurers, and more) to draw on when authoring world/scenario/enemy
> decks, characters, and tutorials. Non-canonical: nothing here binds the rules. Names
> become canonical only when a card in `booklet.ron` (or the Spec) actually uses them.
>
> Tone target: **generic fantasy** (per
> [notes/world-and-progression.md](notes/world-and-progression.md#the-setting)) — grim
> but not grimdark, evocative without lore commitments.

## How to use this

- **Grab and reuse.** Pull a name from here rather than inventing one each time, so
  the setting stays consistent.
- **Mark what's been used.** When a name is committed to a card/scenario, add a
  `— used by <card/scenario>` note beside it so we don't double-assign.
- **Grow it.** Add new sections (regions, enemies, factions, items) as the need arises.
