# Grand Archive — Comprehensive Rules (mirror)

A **verbatim local mirror** of the Grand Archive TCG Comprehensive Rules,
kept as external reference material. Grand Archive is not a game built in this
repo — this folder exists so its rules can be studied and cited offline while
designing the card-table app.

- **Source:** <https://rules.gatcg.com/>
- **Version:** Comprehensive Rules v1.1.1
- **Retrieved:** 2026-07-06
- **Format:** the site's own per-page Markdown (each page also lives online at
  the same path with `.md` appended, e.g.
  <https://rules.gatcg.com/general-rules.md>).

> Copyright of the rules text belongs to Weebs of the Shore / Grand Archive.
> Mirrored here unmodified for reference only. To refresh, re-run the download
> against `https://rules.gatcg.com/llms.txt` (the site's machine-readable index).

## How this mirror is laid out

The directory structure mirrors the site's URL paths exactly. A section that has
sub-pages is both a file (its overview) and a folder (its children) — e.g.
`general-rules.md` alongside the `general-rules/` folder.
