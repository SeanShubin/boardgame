# Game Zones - Material Deck

#### General Rules:

1. The Material Deck is a private zone owned by a player. Each player has their own Material Deck zone.&#x20;

{% hint style="info" %}
The Material Deck zone is referred to by the term “material deck.”
{% endhint %}

2. A player’s starting material deck starts the game in the Material Deck zone.
3. Cards in the material deck must be kept in a single, uniform pile.
4. The material deck must have at least one Level 0 champion card.
5. Constructed formats require that the material deck only contain unique copies of cards.
6. Players may look at their own material deck at any time.
